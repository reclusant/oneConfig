# **oneConfig**



## 一、简介

【**大概私人用**】目录和文件的简单管理

因为是个长期更新的项目，所以放在github上，不过天知道要搞到什么时候🙏

大概就是，把俺的windows文件分类好，并且给各个软件的resource做个backup

未来等俺学成了（）就整个脚本啥的一次安装
